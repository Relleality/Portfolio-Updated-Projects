2026

INVOKE Rice Blast Disease

• INVOKE Rice Blast Disease is an AI-powered web-based system designed for the early detection of rice blast disease in Philippine rice fields. The project utilizes ResNet50 for feature extraction and a Linear Support Vector Machine (SVM) for accurate classification of infected and healthy rice leaves. To improve detection precision, image segmentation techniques were implemented to isolate disease-affected areas. The system also features a user-friendly interface that allows users to upload leaf images and receive real-time analysis, along with a rule-based decision engine that provides actionable recommendations for disease management. This project aims to support farmers and researchers by enabling early intervention and reducing potential crop loss, ultimately contributing to improved agricultural productivity.
